자동 실행

        
    
          
  • 이 창을 닫으면 예약 발행도 함께 종료됩니다. PC를 켜 둔 채 자동으로 돌리려면 이 프로그램을 계속 실행해 두세요.
    • 
          
  • 예약은 매 분 검사하며, 각 계정에 설정한 로컬 시각(HH:mm)에 맞춰 실행됩니다. 기간·하루 횟수는 설정 JSON을 따릅니다.
    • 
          
  • 창 없이 돌리려면 npm run schedule 또는 작업 스케줄러에 --once 로 한 번씩 실행하는 방식을 쓸 수 있습니다.
    • 
        

      


      

        이 프로그램은 어떤 exe인가요?
        

          
이 파일은 Windows에서 더블클릭으로 실행되는 일반 실행 파일(.exe)입니다. 다른 프로그램처럼 배포·설치하면 됩니다.

          
화면은 웹 기술로 만들어진 창이지만, 배포 형태는 일반 데스크톱 앱과 같습니다. (개발자용 이름: Electron)
